## Clone Repo

To get a local copy up and running follow these simple steps:

Open your terminal and run the following commands:

   - `git clone git@github.com:francisuloko/my-portfolio.git`

   - `cd my-portfolio`

Open up with Live Server
   
   - `Alt+L Alt+O`
